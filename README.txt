INDICACIONES DE SEGUNDA PARDE DE PROYECTO POKEMONS

* Dentro del directorio /js se encuentran subdirectorios los cuales son:

    - Directorio /charts en el cual se encuentra: 
        * libreria charts(para manejo de pokemos con detalles) archivo de ejecución chart.html || manejo datos chart.js 
        * libreria charts(para grafica de datos municipios) archivo de ejecución municipis.html || manejo de datos municipis.js

    - Directorio /data en el cual se encuentran:
        * archivos json que nos brindan los datos a utilizar

    - Directorio /maps en el cual se encuentran:
        * libreria de GoogleMaps(para manejo de coordenadas de municipios) archivo de ejecución maps.html || manejo de datos maps.js

    - Directorio /pokemons en el cual se encuentran:
        * archivo de ejecucion se encuentra en la raiz index.html
        * pokemon.js (primera version del proyecto manejo de arrays)
        * pokemonAjax.js (primera version del proyecto manejo de arrays)
        * pokemonObject.js (segunda y actual version de manejo de objetos para pokemons)
    
    ANOTACION:el archivo pokemonObject.js solo contiene la forma de como obtener los datos de los pokemons de los json e ir creando 
    objetos de dichos pokemos con una clase usada de plantilla para los objetos y al final muestro los datos por consola dentro de un array
    que contiene todos los objetos pokemon.
